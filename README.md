# MindX Support AI

## Tech Stack
- Java
- Spring Boot
- MySQL
- React (if added later)

## Features
- Create support tickets
- Store user messages
- View all tickets
- View ticket details with messages

## API Endpoints

### Create Ticket
POST /tickets

### Get All Tickets
GET /tickets

### Get Ticket Details
GET /tickets/{id}

## How to Run

1. Clone repository
2. Open in IntelliJ
3. Configure database in application.properties
4. Run Spring Boot application
